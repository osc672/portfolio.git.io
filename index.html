Project: ONLINE FAKE LOGO DETECTION SYSTEM.

View More

Objective: The primary objective of this project is to develop an advanced system for detecting fake logos on online platforms. The system aims to identify counterfeit logos accurately and efficiently, helping businesses protect their brand integrity and consumers avoid fraudulent products.

Project Type: Developed a Convolutional Neural Network (CNN) model using Python in Google Colab and supports Windows 8/10 system with GPU acceleration.

Layers Used: Included Conv2D, MaxPooling2D, Dropout, and Batch Normalization layers.

Training Data: Trained on 12,400 grayscale images resized to 256x256 pixels.

Objective: Accurately distinguished between original and counterfeit logos.

Accuracy: Achieved high accuracy through intensive training and validation on a weighted dataset.

Evaluation Techniques: Used visualization and predictive analysis on test images to ensure reliable logo detection performance.
